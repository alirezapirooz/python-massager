
📱 #Social Media App (Python CLI Project)

This is a simple social media simulation app built with Python.
It runs in the terminal (CLI) and allows users to sign up, log in, create posts, send messages, and manage friend connections.

🚀 ##Features
👤 User registration and login system
📝 Create and view posts (timeline feed)
💬 Send and receive private messages
🤝 Friend request system (add/accept friends)
💾 Persistent data storage using CSV files
🧠 Data automatically saves after every action
🔐 Admin account created by default
🛠️ Technologies Used
Python 3
CSV module (file-based database)
datetime module
Object-Oriented Programming (OOP)
📂 Data Storage

The app uses CSV files to store data:

users.csv → user accounts
posts.csv → posts
messages.csv → private messages
friends.csv → friendships
friend_requests.csv → pending friend requests

All data is saved automatically after each action.
